Creating A New Account

  • Creating a new account is very easy. Just select your account category and fillup input boxes which ever relevant.

  • During account creation, for Password field you can ask for Password suggestions or input your own Password. This is useful during changing existing password or new account creation.



Email of User Key

  • Since we do not store your user key, if you lose it, your data is unlockable. So in case you forget this key, we send an decryption link to your email everytime you create a new account.

  • This email is secured. Even if you forward this email to anybody else, the decryption link will not work. To make it work they have to login to Lock Essential using your account password.



Unlock Locked Accounts

  • If you input wrong userkey, Lock Essential will display all of your locked accounts.

  • To unlock it you have to input valid user key.
