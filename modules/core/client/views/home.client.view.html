Secures Your Data In 3 Layers

User Password Layer

Your data is protected by your account password using industry standard technology (PassportJs Local Strategy, Google/Facebook Login Strategy)

User Key Layer

This is your master key. Data is encrypted by this key using 'AES' technology. Since we do not store this key, your data is technically protected from everybody. If you forget this key, no worries. Check your email!

Server Key Layer

This is another server side protection using custom server key using 'AES' encryption technology. So in case your user key is compromised, this will give you extra protection. You do not have to memorize this key. It is an automated protection.
